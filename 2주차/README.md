# 2주차 위클리 챌린지 회고
## 학습 회고
FastAPI 다루는 것이 처음이라 위클리 챌린지 2.1 HTTP REST API 설계 및 구현에서 이미 너무 많은 시간이 걸렸다. 강의를 들은 직후에 개념만 복습하지 말고 다른 다양한 예제들을 따라하면서 강의에서 배운 개념들을 이렇게 적용해서 이용할 수 있구나를 미리미리 해야했다는 생각이 들었고 그러지 못해서 아쉬웠다. 그래서 과제 시작이 늦어진 느낌이 많이 들었고 시간이 매우 빠듯하게 느껴졌다. 다음에는 개념을 모두 이해하고 나서 위클리 챌린지에 들어서는 것이 아니라 바로바로 적용시킬 수 있도록 개념 정리와 병행하며 학습이 필요할 것 같다.

## 과제 코드 회고
예제를 따라하면서 강의 시간에 배운 내용을 이렇게 접목시키는 거구나를 느꼈고 REST API의 동작 과정에 대해 이해하게 된 시간이었다. 또한 AI에게 거의 물어보지 않고 오로지 예제 코드를 이해하고 적용시키려고 노력했습니다. 평소에도 AI를 사용하는 습관은 없어서 이게 좋은 건지 나쁜건지는 모르겠지만 아직은 최대한 AI에게 물어보거나 시키는 것은 자제하려고 한다.
