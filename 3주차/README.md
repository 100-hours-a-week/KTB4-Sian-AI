### 미니퀘스트 회고
- 이번 데이터 분석과 시각화의 미니퀘스트를 수행하며 최대한 0단계나 1단계 코딩을 지키기 위해 공식 문서 위주로 읽고 문제를 해결하려고 했습니다. 웬만한 퀘스트들은 강의 교재의 내용만으로 수행이 가능했으나 몇몇 문제는 교재에 나와있지 않은 기능들을 사용해야 했고 그 과정에서 공식문서만을 보고 내가 원하는 기능을 서치해내는 것은 꽤나 어려운 일이란 것을 느꼈습니다. 특히 특정 키워드로 서치하게 되면 그 서치 키워드가 설명에 나타나 있는 수많은 기능들이 모두 검색 결과에 나와서 검색이 어려웠던 것 같습니다. 수많은 검색 결과 안에서 원하는 기능을 찾아내고 골라내는 능력이 필요하다고 느꼈으며 그와 더불어 서치할 때 단순히 키워드만 넣는 것이 아니라 좀 더 구체적으로 찾아내는 방안 또한 고민해야겠다는 생각을 했습니다.

### 개선 방안
- 멘토링 시간에 답변 해주셨듯이 공식 문서 안에서 내가 원하는 키워드를 걸러내고 찾아낼 수 있는 그런 프로그램을 직접 짤 수 있으면 편리하고 좋을 것 같기에 어떻게 하면 좋은 검색 방법을 고안해낼 수 있을지 이번 개인 학습 기간동안 고민하여 만들어내고 싶습니다.
